Create a high-fidelity desktop web app prototype for a platform called "EduNest".

EduNest is a tutor–parent matching platform in Vietnam.
It helps parents find suitable tutors based on subject, grade level, price, rating, location, and learning format (online/offline).

Target users:
- Parents (30–50 years old)
- Tutors (university students, teachers)

Design style:
- Clean, modern, minimal
- Trustworthy and professional (EdTech startup style)
- Primary color: #2F80ED
- Secondary color: #27AE60
- Light background (#F8FAFC or white)
- Soft rounded cards
- Subtle shadows
- Spacious layout
- Clear Vietnamese UI text

Layout:
Desktop size: 1440px width
Responsive grid system
Left navigation (optional for dashboard)
Top navigation for main site

Include the following pages:

1. Landing Page
- Top navbar:
  - Logo (EduNest)
  - Trang chủ
  - Tìm gia sư
  - Trở thành gia sư
  - Đăng nhập
  - Đăng ký (highlighted button)
- Hero section:
  - Headline: "Tìm gia sư uy tín cho con bạn chỉ trong vài phút"
  - Subtext: "Minh bạch thông tin – Giá rõ ràng – Đặt lịch dễ dàng"
  - Search bar (Subject | Grade | Location | Search button)
- How it works (3 steps)
- Featured Tutors section (card grid 3–4 per row)
- Footer with contact info

2. Tutor Listing Page
- Left sidebar filters:
  - Môn học
  - Lớp
  - Giá
  - Hình thức học
  - Đánh giá
- Main content:
  - Tutor cards in grid layout
    - Avatar
    - Name
    - Rating
    - Subjects
    - Price per session
    - Location
    - Button: "Xem chi tiết"

3. Tutor Detail Page
- Large profile section
- Left: Avatar + basic info
- Right: Booking card (sticky)
  - Price
  - Select date
  - Select format
  - Button: "Đặt lịch"
- Tabs:
  - Giới thiệu
  - Kinh nghiệm
  - Đánh giá
- Reviews list UI

4. Dashboard (Parent)
- Sidebar:
  - Tổng quan
  - Lịch học
  - Gia sư đã lưu
  - Tin nhắn
  - Hồ sơ
- Main content:
  - Upcoming sessions
  - Recent messages
  - Quick stats

5. Chat Page
- Two-column layout
  - Left: Conversation list
  - Right: Chat window

Make it interactive:
- Add clickable navigation between pages
- Create consistent component system (buttons, cards, input fields)
- Use modern SaaS dashboard design inspiration
- Layout similar to Airbnb or modern EdTech platforms

Typography:
- Clean sans-serif (Inter or similar)
- Clear hierarchy (H1, H2, body text)

Make it look like a real startup MVP ready for investor presentation.
