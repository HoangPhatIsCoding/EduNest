Create a high-fidelity desktop UI design for an EdTech platform called "EduNest".

EduNest is a platform in Vietnam that connects parents with tutors.

Design requirements:

* Desktop layout (1440px width)
* Clean, modern, minimal EdTech style
* Primary color: #2F80ED
* Secondary color: #27AE60
* Light background (#F8FAFC)
* Soft rounded corners
* Subtle shadows
* Modern sans-serif font (Inter)

Use Vietnamese for UI text.

Create only the following screens:

1. LOGIN PAGE

Layout:
Split screen layout.

Left side:

* Illustration related to education or tutoring
* EduNest logo
* Short tagline: "Kết nối gia đình và gia sư uy tín"

Right side:
Centered login card with:

* Title: "Đăng nhập"

Fields:

* Email
* Mật khẩu

Buttons:

* Đăng nhập (primary button)
* Đăng nhập với Google

Links:

* Quên mật khẩu?
* Chưa có tài khoản? Đăng ký

---

2. REGISTER PAGE

Layout:
Same style as login page.

Title:
"Đăng ký tài khoản"

Role selection at top:

* Phụ huynh
* Gia sư

Form fields:

* Họ và tên
* Email
* Mật khẩu
* Xác nhận mật khẩu

Buttons:

* Tạo tài khoản (primary)

Footer text:
"Đã có tài khoản? Đăng nhập"

---

Interaction:
Make the link "Đăng ký" on the Login page navigate to the Register page.
Make the link "Đăng nhập" on the Register page navigate back to the Login page.

Use modern UI spacing and reusable components (inputs, buttons, cards).
The result should look like a real startup MVP authentication system.
