Add new desktop screens for the EduNest platform (1440px width).

EduNest is an EdTech platform in Vietnam that connects parents with tutors and helps manage tutoring sessions.

Design Style:

* Clean modern EdTech design
* Primary color: #2F80ED
* Secondary color: #27AE60
* Background: #F8FAFC
* Rounded cards
* Soft shadows
* Spacious layout
* Font: Inter or modern sans-serif
* Vietnamese UI text

Create the following pages:

---

1. TUTOR HOME PAGE (Trang chủ gia sư)

Layout: Sidebar + Main content dashboard.

Sidebar Menu:

* Trang chủ
* Lịch dạy
* Tài liệu
* Tin nhắn
* Học sinh của tôi
* Hồ sơ
* Đăng xuất

Main Dashboard Content:

Top section:
Greeting message:
"Xin chào, [Tên gia sư]"

Stats cards:

* Buổi học hôm nay
* Tổng số học sinh
* Lịch học tuần này

Section: Lịch dạy hôm nay
Table or card list showing:

* Student name
* Subject
* Time
* Button "Xem chi tiết"

Optional section:
"Câu hỏi luyện tập cho học sinh"

Button:
"Tạo câu hỏi"

Simple form to create a question:

* Question title
* Question content
* Upload file
* Assign to student

---

2. LESSON DETAIL PAGE (Chi tiết lịch học)

Page title:
"Chi tiết buổi học"

Layout: Two-column content.

Left section:
Lesson information card:

* Student name
* Subject
* Date
* Time
* Learning format (Online / Offline)

Lesson notes section:
Text area for tutor notes.

Right section:

Attendance card:

* Student status
  Options:
* Có mặt
* Vắng mặt

Score input:
Field for entering student score.

Buttons:

* Lưu thông tin
* Kết thúc buổi học

---

3. MATERIAL PAGE (Trang tài liệu)

Page title:
"Tài liệu học tập"

Top actions:

* Upload tài liệu button
* Search bar

Material cards list:
Each card includes:

* File icon
* File name
* Subject
* Upload date
* Download button

Optional filters:

* Subject
* Grade level

---

4. CHATBOX UI

Floating chat widget that appears at bottom-right of the screen.

Closed state:
Small chat icon.

Open state:
Chat window with:

Top bar:

* Tutor or parent name
* Status indicator

Messages area:

* Left aligned messages
* Right aligned messages

Bottom input area:

* Text input
* Send button
* Attachment icon

Design similar to modern support chat widgets.

---

5. PARENT MANAGEMENT PAGE

Page title:
"Quản lý việc học"

Layout: Dashboard style.

Sections:

Student Overview Card:

* Student name
* Grade
* Current tutor

Attendance Section:
Table showing:

* Date
* Subject
* Tutor
* Attendance status

Score Section:
Table showing:

* Subject
* Score
* Tutor comment

Progress Chart:
Simple progress visualization of student performance.

Buttons:

* Xem chi tiết buổi học
* Nhắn tin với gia sư

---

Interaction:

All pages should follow the same design system as EduNest.

Use reusable components:

* Cards
* Tables
* Buttons
* Inputs

Add hover states and clickable navigation between pages.
