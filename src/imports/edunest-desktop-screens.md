Add two additional desktop screens for the EduNest platform.

EduNest is an EdTech platform in Vietnam that connects parents with tutors.

Design Style:

* Desktop layout (1440px)
* Clean, modern EdTech design
* Primary color: #2F80ED
* Secondary color: #27AE60
* Background: #F8FAFC
* Rounded cards
* Subtle shadows
* Modern sans-serif font (Inter)

Use Vietnamese UI text.

---

1. TUTOR REVIEW PAGE

Page title:
"Đánh giá gia sư"

Layout:
Centered content with tutor information at the top.

Tutor Info Card:

* Tutor avatar
* Tutor name
* Subjects taught
* Rating summary (example: 4.8 ⭐ from 25 reviews)

Review Form:

Star rating input (1–5 stars)

Fields:

* Tiêu đề đánh giá
* Nội dung đánh giá

Example placeholder text:
"Chia sẻ trải nghiệm học với gia sư này..."

Buttons:

* Gửi đánh giá (primary button)

Below the form:
Reviews list

Each review card includes:

* Parent avatar
* Parent name
* Rating stars
* Review title
* Review content
* Date

---

2. CHAT PAGE (PARENT – TUTOR MESSAGING)

Layout:
Two-column chat interface.

Left Panel (Conversation List):

* List of tutor conversations
  Each item includes:
* Tutor avatar
* Tutor name
* Last message preview
* Timestamp

Right Panel (Chat Window):

Top bar:

* Tutor avatar
* Tutor name
* Status: "Đang hoạt động"

Chat messages area:
Message bubbles for both users

* Parent messages aligned right
* Tutor messages aligned left

Bottom input area:

* Text input field
* Send button
* Optional attachment icon

Example placeholder:
"Nhập tin nhắn..."

---

Interaction:
Clicking a tutor in the conversation list opens the chat in the right panel.

Design should look similar to modern messaging interfaces like Messenger or Slack but adapted for an education platform.

Maintain consistent design system with existing EduNest screens.
